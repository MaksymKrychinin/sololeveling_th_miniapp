## 🚀 Ready to Run!

### Quick Start:

```bash
# 1. Setup (automatic)
./setup.sh

# 2. Or manual
pnpm install
pnpm docker:dev
cp .env.example .env
# Edit .env with your Telegram bot token
pnpm db:migrate
pnpm db:seed

# 3. Start development
pnpm dev

# Apps will run on:
# - Web: http://localhost:3000
# - API: http://localhost:3001
# - Bot: Running in background
```

### Test API:

```bash
# See API_DOCS.md for full documentation
curl http://localhost:3001/api/v1/health
```
